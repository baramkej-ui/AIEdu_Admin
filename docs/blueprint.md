# **App Name**: EduQuiz 플랫폼

## Core Features:

- 사용자 인증: 역할 기반 커스텀 클레임(관리자, 교사, 학생)을 사용하는 보안 이메일/비밀번호 로그인
- 역할 기반 라우팅: 사용자 역할에 따라 로그인 후 자동 리디렉션 (관리자 → /dashboard, 교사 → /students, 학생 → /problems)
- 문제 관리: 교사와 관리자는 질문, 옵션, 답변 및 난이도를 사용하여 문제를 생성, 업데이트 및 삭제할 수 있습니다.
- 학생 진행 상황 추적: 해결된 문제, 정확성, 소요 시간을 포함한 학생 진행 상황을 추적하고 Firestore에 저장합니다.
- 문제 생성: 생성적 AI 도구를 사용하여 새로운 문제 세트를 제안합니다. AI 도구는 문제 수준을 적절하게 조정하기 위해 이 학생에 대해 이전에 해결된 문제와 최근 문제 정보를 고려합니다.

## Style Guidelines:

- 기본 색상: 신뢰와 지능을 나타내는 짙은 파란색 (#3F51B5).
- 배경 색상: 차분하고 집중된 환경을 위한 기본 색상의 매우 가벼운 채도의 색조인 밝은 파란색 (#E8EAF6).
- 강조 색상: 정의를 부여하기 위해 채도와 밝기에 약간의 변화가 있는 기본 색상과 유사한 색상인 보라색 (#7E57C2).
- 본문 및 제목 글꼴: 가독성을 위한 깨끗하고 현대적인 모양의 산세리프체인 'Inter'.
- 코드 글꼴: 코드 스니펫을 표시하기 위한 'Source Code Pro'.
- 이해하기 쉽도록 탐색 및 작업에 명확하고 교육적인 아이콘을 사용하십시오.
- 장치(데스크톱, 태블릿, 모바일)에서 접근성을 위한 반응형 레이아웃 디자인.
- 로딩, 성공 또는 오류 상태를 나타내는 사용자 경험을 향상시키기 위한 미묘한 전환 및 피드백 애니메이션.
